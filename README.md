# Insurance Claims Analysis

## Overview
Analysis of 50,000+ insurance records to identify 
key factors affecting claim rates.

## Tools Used
- PostgreSQL — data cleaning and querying
- Python (pandas, matplotlib) — analysis and visualization
- Power BI — interactive dashboard

## Key Findings
- Only 6.4% of customers made claims
- Newer vehicles have higher claim rates
- C18 region has highest claim rate at 10%+

## Files
- cleaning.sql — data cleaning queries
- analysis.sql — SQL analysis
- python/ — Python analysis notebook
- powerbi/ — Power BI dashboard
